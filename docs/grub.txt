Menu will appear if you press and hold Shift during loading Grub, 
if you boot using BIOS. When your system boots using UEFI, press Esc.

For permanent change you'll need to edit your 
/etc/default/grub 
file -- place a "#" symbol at the start of line 
GRUB_HIDDEN_TIMEOUT=0.

Save changes and run sudo update-grub to apply changes.
